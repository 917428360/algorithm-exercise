# 算法面试准备

## Coding Style - 代码风格

- 代码块可为三大块：异常处理，主体，返回
- 代码风格(可参考Google的编程语言规范)
    - 变量名的命名(有意义的变量名)
    - 缩进(语句块)
    - 空格(运算符两边)
    - 代码可读性(即使if语句只有一句也要加花括号)

面试过程要有适当的沟通：
1. 听到问题后要讲自己的理解
2. 与面试官沟通题目的细节
3. 告诉面试官自己的想法或算法

如何刷题？
1. 总结归类相似题目
2. 找出适合同一类题目的模板程序
3. 对基础题熟练掌握
